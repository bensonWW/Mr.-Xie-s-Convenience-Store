# Technology Stack

## Backend
-   **Framework**: Laravel 12.x
-   **Language**: PHP 8.2+
-   **Database**: MySQL 8.0
-   **Search Engine**: SQL `LIKE` with Index (Meilisearch removed).
-   **Cache/Queue**: Redis.
-   **Authentication**: Laravel Sanctum (API Token).
-   **Testing**: PHPUnit / Pest.

## Frontend
-   **App Name**: `xie_vue`
-   **Framework**: Vue.js 3
-   **State Management**: **Vuex** (Confirmed).
-   **Router**: Vue Router.
-   **HTTP Client**: Axios.
-   **Styling**: **Tailwind CSS** (Confirmed).

## DevOps & Infrastructure
-   **Containerization**: Docker & Docker Compose.
-   **Server**: Nginx (via Docker).
-   **Orchestration**: Kubernetes (files exist in `k8s/`, but primary focus is `docker-compose` for dev).

## Tools
-   **Package Manager**: Composer (PHP), NPM (JS).
-   **Version Control**: Git.
